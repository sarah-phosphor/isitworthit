# Flags

Country flag SVGs for the 48-team field, keyed by ISO 3166-1 alpha-2 (home nations use `gb-eng` / `gb-sct`).
Sourced once from [flag-icons](https://github.com/lipis/flag-icons) (MIT) and committed locally — referenced as `/flags/<iso>.svg`, never hot-linked.
Mapping from ESPN/FIFA codes lives in `src/lib/flags.ts`.
